# SemperMove Frontend

Flutter-приложение для отслеживания фитнес-активности и соревнований.

## Запуск

```bash
flutter pub get
flutter run
```

## Функции

- Регистрация и авторизация
- Отслеживание ежедневной активности (шаги, отжимания, приседания, планка, вода)
- Дуэли с другими пользователями
- Статистика и история прогресса
- Профиль пользователя

## Технологии

- Flutter 3.9.2+
- Provider (State Management)
- HTTP (API запросы)
- FL Chart (Графики)
- Shared Preferences (Локальное хранилище)
